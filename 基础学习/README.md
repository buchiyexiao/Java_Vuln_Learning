### 访问控制权限

访问控制权限主要包含private（私有）（get set方法），package（包访问权限），protected（子类访问权限），public（公共访问权限）

|            | private | package | protected | public |
| :--------: | :-----: | :-----: | :-------: | :----: |
| 同一个类中 |    √    |    √    |     √     |   √    |
| 同一个包中 |         |    √    |     √     |   √    |
|   子类中   |         |         |     √     |   √    |
| 全局范围内 |         |         |           |   √    |
